大四一些总结

2024/5/2 生活

一些杂七杂八的

一晃悠闲大四快过完了，感觉这一年里自己似乎并没有做太多东西，也没有成为我想要的样子。先说一下我这段光阴里做的事情以及一些感悟：

做到的新事情：

  • 跟随跋涉者参加腾格里徒步 / 参加梁子湖露营 / 爬康王寨和棋盘石 / 登顶大别山主峰白马尖 / 恩施泡温泉 / 游南昌&广州
  • 学会阅读论文并找到一些新idea,当然代码能力还有所欠缺
  • 象棋下到业6-1，实际水平可能差不多业7（虽然好像没有什么用qaq）
  • 在xhs发post并火了一把，加了一些新朋友

没能做到的事情：

  • 丢掉了一些好习惯，没有每天健身锻炼而且开始晚睡晚起了，导致身上湿气比较重脾胃功能相对较虚。
  • 没有主动联系人，让自己和很多朋友有点渐行渐远；同样本学期我既没有怎么发朋友圈也没有去追女生
  • 科研还有一些项目自己并没有很好的落实，畏难情绪严重而且喜欢沉迷舒适区，导致好多计划没有实施，欠了自己一些账。
  • 毕设拖延情况严重，在最后关头才搞完，给自己一个挺大的教训。

那么在剩下来的差不多4个月的空余时间里面，作为日常喜欢玩喜欢制定计划但是从来都不喜欢完成的P人，我应该抓住一些东西不要让自己这一年留下太多遗憾

任务清单

  • 长期任务：

  • 1、在学校里面尝试健身房一周去三次，并每周至少一次游泳

  • 2、早睡早起，早上8：15起床，晚上12：30睡觉

  • 3、清理wolai上学习计划

  • 短期任务（5~8月的毕业季）：

  • 1、拍一次毕业照，在毕业典礼时主动联系以前同学一起拍照

  • 2、找之前初/高中同学/学长学姐聚会（还欠cgx,hjm,zys一顿饭）

  • 3、看望初高中老师

  • 4、捞人去一趟毕业旅行（5~6月），建议去新马泰

  • 5、投一次实习

  • 6、去一趟酒馆与ktv

  • 7、去听一次讲座

  • 8、完整听一次草坪音乐会

  • 9、尝试自己完整完成一个感兴趣的项目并上传github

  • 10、至少完成一门cs229/cs224N/CS236并写完作业
